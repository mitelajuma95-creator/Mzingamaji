1. Nenda netlify.com → Jiandikishe bure
2. Bonyeza "Add new site" → "Deploy manually"
3. Buruta faili ya MzingaMaji.html ndani
4. Utapata link kama: mzinga.netlify.app
5. Fungua link hiyo kwenye simu yako ✅
🤖 Hatua 2A — Android (Chrome)
1. Fungua link ya Netlify kwenye Chrome
2. Bonyeza dots 3 ⋮ juu kulia → "Add to Home screen"
3. Bonyeza Install — tayari ✅
🍎 Hatua 2B — iPhone (Safari)
1. Fungua link ya Netlify kwenye Safari
2. Bonyeza ☐↑ (Share) chini
3. Tembeza → bonyeza "Add to Home Screen"
4. Bonyeza Add — tayari ✅
